# Famago - Industrial Design Furniture for Modern Spaces


Welcome to **Famago**, your go-to source for **high-quality industrial design furniture** that brings modern elegance and functionality to any space. Whether you are designing your home or office, Famago's furniture seamlessly blends **minimalism with practicality**, making sure your space looks stylish and works efficiently.

Our website is built using cutting-edge technologies like **Astro**, **Tailwind CSS**, and **Preline UI**, offering a fast, smooth, and visually stunning shopping experience.

**[View Live Demo](https://famago.cl)**

## Features

- **Astro-Powered**  
  Our site is powered by **Astro**, a modern static-site generation framework that ensures excellent performance and fast load times.

- **Tailwind CSS**  
  With **Tailwind CSS**, we bring a utility-first, responsive design that allows for quick and easy UI customization, giving your website a polished look.

  We've incorporated **Preline UI** for prebuilt components that are both aesthetically pleasing and functional, enhancing your site with elegant design elements.

## Installation

To get started with Famago locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/mearashadowfax/Famago.git
   ```

2. Navigate into the project directory:

   ```bash
   cd Famago
   ```

3. Install the required dependencies:

   ```bash
   npm install
   ```

4. Run the development server:

   ```bash
   npm start
   ```

Your local Famago site will now be running at `http://localhost:3000`.
